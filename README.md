# spt

### Minimum requirements:
  - node: >= 7.6
  - macOs
  
### Install
    - $ git clone git@github.com:brnbp/spotify-for-terminal.git && cd spotify-for-terminal
    - $ yarn

    

some commands

$ node spt play
$ node spt pause
$ node spt next
$ node spt prev
$ node spt status
$ node spt open
$ node spt close
$ node spt --help
